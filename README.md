# WebApp_G12

## Run Method
In terminal you should follow the following instructions:

### Run Frontend
```bash
cd frontend
npm install
npm run dev
```

### Run Backend
```bash
cd ..
cd backend
python3 main.py
```

### Run Database
```bash
cd ..
cd db
```
- Copy `.env.example` to another file named `.env` under db directory.
```bash
npm install
node index.js
```
